对于游戏引擎版本的文件，进行版本控制可能需要 Helix Core 忽略某些文件，例如单个用户设置、缓存等。

设置和创建忽略文件 P4 ignore

设置步骤为：

  • 创建ignore.txt，写好ignore内容-保存-上传p4根目录
  • 运行bat，并检查是否过滤成功

P4V_设置.ignore

过滤忽略无需同步的文件。

2.用下面代码创建一个P4ignore.bat ， 然后运行.

::: code-tabs#shell

@tab P4ignore.bat

@echo off
set workdir=%~dp0
set result=%workdir%p4ignore.txt
echo %result%
p4 set P4IGNORE="%result%"
pause

:::
