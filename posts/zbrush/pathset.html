Zbrush Path Set

Megestus2024年2月27日ZbrushZbrush
此页内容
  • Path
  • Goz set

Path

  • 插件安装路径：把解压文件放到 C:\Program Files\Pixologic\ZBrush 2019\ZStartup\ZPlugs64
  • 笔刷放置路径：C:\Program Files\Pixologic\ZBrush 2019\ZData\BrushPresets
  • 材质球放置路径：C:\Program Files\Pixologic\ZBrush 2019\ZData\Materials\MatCap
  • UI放置路径：C:\Program Files\Pixologic\ZBrush 2019\ZStartup\UserInterfaceLayouts
  • 灯箱笔刷路径: C:\Program Files\Pixologic\ZBrush 2019\ZBrushes

Goz set

GoZ文件在 C:\Users\Public\Pixologic\GoZApps\Maya

  • 复制shelf_GoZBrush. mel 到 Program Files\Autodesk\Maya20XX\scripts\shelves
  • 复制GoZScript. mel 到 C:\Program Files\Autodesk\Maya20XX\scripts\startup
  • 复制gozMaya.mll （对应版本号）到 C:\Program Files\Autodesk\Maya20XX\bin\plug-ins

注意：gozMaya.mll文件夹必须安装对应的Maya版本，低于电脑安装的maya版本则无效。

如果maya没有goz图标就把shelf_GoZBrush. mel 复制到
C:\Users\Administrator\Documents\maya\20XX\zh_CN\prefs\shelves

如Zb在使用Goz的过程中会出现，每Goz一次，就会打开一个新的Maya。 [1]
重装Maya，或者重装Zbrush都不行。清理Goz的历史记录、重新设置Goz路径，也不行。

出现以上情况可以通过以下方法：

在这个路径：C:\Users\Public\Pixologic\GoZApps\Maya，找到"userSetup.mel" 这个文件
然后复制到C:\Users\管理员\Documents\maya\20XX\scripts 路径下。
（注：管理员是你自己的主机管理员名称）
复制完成后，即可生效，无需重启。

  1. 采集于 Blibili-wanshi1988 ↩︎
